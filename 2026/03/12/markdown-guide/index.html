Markdown 常用语法标准指南

这是一份旨在帮助你快速掌握 Markdown 排版的参考手册。Markdown 是一种轻量级标记语言，它允许你使用易读易写的纯文本格式编写文档。

1. 标题 (Headings)

使用 # 号开头，1 到 6 个 # 分别对应一到六级标题。

一级标题 (H1)

二级标题 (H2)

三级标题 (H3)

四级标题 (H4)

五级标题 (H5)
六级标题 (H6)

2. 文本格式 (Text Formatting)

  • 加粗：**这是加粗文字** 或 __这是加粗文字__
  • 斜体：*这是斜体文字* 或 _这是斜体文字_
  • 粗斜体：***加粗且斜体***
  • 删除线：~~这是删除线~~

3. 列表 (Lists)

无序列表

使用 -、+ 或 * 加上空格。

  • 项目一
  • 项目二
    • 子项目（缩进两空格）

有序列表

使用数字加上 . 和空格。

  1. 第一步
  2. 第二步
  3. 第三步

任务列表 (Task Lists)

  • 已完成任务
  • 待办事项
  • 计划中

4. 链接与图片 (Links & Images)

  • 行内链接：[Google](https://www.google.com)
  • 带标题的链接：[百度](https://www.baidu.com "搜索引擎")
  • 插入图片：![图片描述](图片链接地址)

5. 引用与分割线 (Quotes & Rules)

引用

使用 > 符号。

这是一个块状引用。

也可以嵌套引用。

分割线

使用三个或更多的 ---、*** 或 ___。

6. 代码 (Code)

行内代码

使用单反引号包裹：print("Hello World")

代码块

使用三个反引号（```）包裹，并可以指定语言进行高亮。

def greet(name):
    return f"Hello, {name}!"

print(greet("Markdown"))
